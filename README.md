# Box — متجر إلكترونيات

مشروع React + Vite + Tailwind جاهز للتشغيل والنشر.

## التشغيل محليًا

يتطلب [Node.js](https://nodejs.org) مثبت على جهازك (نسخة 18 أو أحدث).

```bash
npm install
npm run dev
```

بعدها افتح الرابط اللي يظهر بالتيرمنال (عادة `http://localhost:5173`).

## البناء للنشر (Build)

```bash
npm run build
```

هذا يولّد مجلد `dist/` يحتوي كل ملفات الموقع جاهزة للرفع.

## النشر (Deploy)

أسهل الطرق:

### Vercel
1. ارفع المشروع على GitHub.
2. روح [vercel.com](https://vercel.com) → New Project → اختر الريبو.
3. Vercel يكتشف Vite تلقائيًا، اضغط Deploy.

### Netlify
1. اسحب مجلد `dist/` (بعد تشغيل `npm run build`) مباشرة على [app.netlify.com/drop](https://app.netlify.com/drop).

### أي استضافة ثابتة أخرى
ارفع محتويات مجلد `dist/` بعد البناء إلى أي استضافة تدعم مواقع static (GitHub Pages, Cloudflare Pages, إلخ).

## ملاحظات مهمة

- **لا يوجد باكند حقيقي**: السلة، تسجيل الدخول، ورمز التفعيل كلها تعمل بالكامل من طرف المتصفح (state) بدون قاعدة بيانات أو سيرفر. أي تحديث بالصفحة يفرّغ البيانات.
- **رقم المالك**: معرّف بثابت `OWNER_NUMBER` داخل `src/App.jsx` — غيّره لرقمك الحقيقي إذا لزم.
- **رقم واتساب التواصل**: ثابت `WHATSAPP_NUMBER` بنفس الملف.
- لإضافة باكند حقيقي (حفظ الطلبات، إرسال SMS فعلي لرمز التفعيل، قاعدة بيانات منتجات) تحتاج ربط خدمة خلفية منفصلة (مثل Supabase أو Firebase أو سيرفر خاص).
